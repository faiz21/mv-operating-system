# MV-OS — Machine Vision Operating System

Internal wiki and operating system for the Machine Vision three-agency intelligence framework.

## Overview

MV-OS organizes three agencies — **Machine Vision**, **The Consultant**, and **The Auditor** — each with defined scope, agents, and outputs.

| Stat | Value |
|------|-------|
| Agencies | 3 |

## Agencies

### MV — Machine Vision (Main Agency)
The operating client. Receives solutions, executes briefs, and runs lines of business across Consulting, Software, and Automation SI.

### CON — The Consultant (Advisory Agency)
Designs workflows, synthesizes audit findings, architects data schemas, and builds monitoring UIs for Machine Vision. Staffed by 1 orchestrator + 7 sub-agents across 6 domains.

### AUD — The Auditor (Audit Agency)
Reviews Machine Vision's execution against issued briefs, produces audit findings, detects deviations, and escalates recurrences.
