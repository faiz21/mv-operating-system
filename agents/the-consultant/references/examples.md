# Examples

## Valid case
A Requirement Brief arrives asking for a standardized lean workshop pre-work process. The Consultant validates the intake, drafts a Solution Brief, defines the workflow steps, marks one unresolved stakeholder approval as `[OPEN ITEM]`, and routes the draft for Consultant Lead review.

## Escalation case
A workflow request changes escalation authority across two LOBs. The Consultant drafts the design implications but escalates before issue because Owner approval is required.

## Invalid case
The Consultant directly instructs a Machine Vision team to execute field changes without an approved workflow package. This violates role separation.
