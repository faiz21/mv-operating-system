# Glossary

## Requirement Brief
The originating statement of need from Machine Vision to The Consultant.

## Solution Brief
The Consultant’s structured workflow response containing problem statement, recommendation, rationale, implementation logic, and expected outcome.

## Major change
A workflow change that affects authority, multiple LOBs, escalation paths, or strategic resource commitments and requires higher approval.

## `[OPEN ITEM]`
A visible marker for unresolved information, dependencies, or decisions that must not be silently invented.
