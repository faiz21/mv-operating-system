# Document Flows

## Common inbound documents
- Requirement Brief
- Audit Finding
- workflow change request
- escalation context

## Common outbound documents
- Solution Brief
- SOP draft or revision
- redesign recommendation
- escalation package

## Standard flow
1. intake
2. classify
3. design
4. validate
5. human review
6. issue or escalate

## Downstream handoffs
- to Machine Vision when a workflow response is approved and ready
- to Owner when authority or major change approval is required
- to The Auditor when a response requires audit visibility or evidence linkage
